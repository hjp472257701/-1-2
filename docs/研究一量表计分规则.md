# 研究一：量表计分规则（与线上一致）

本文档与 `server/questionnaireSchema.js`（问卷版本见其中 `version` 字段）及 `server/index.js` 中 `scoreQuestionnaire()` / `scoreReverse()` 的实现一致，供后续论文写作与数据分析对照。

## 1. 通用步骤

1. 每题存库为原始作答值 `raw`（整数，落在该量表的 `min`～`max` 之间）。
2. 若题目 ID 出现在该量表的 `reverseItems` 中，先做反向：  
   **反向后分 = min + max − raw**  
   （`min`、`max` 为该量表在 schema 中的统一上下界。）
3. 未作答的题目不参与该量表汇总（当前端要求全部填答后提交时，完整个案通常为满题）。
4. 按该量表 `scoring.method` 对「反向后或原始的」各题分进行 **求和（sum）** 或 **算术平均（mean）**。

## 2. 各量表明细

| 量表 ID | 量尺 | 反向计分题目 ID | 汇总方式 | 备注 |
|---------|------|-----------------|----------|------|
| PRDS | 1～6 | PRDS02、PRDS04 | **sum** | 5 题；分越高通常表示个人相对剥夺感越高 |
| PMD | 1～7 | 无 | **sum** | 24 题；分越高表示道德脱离倾向越高 |
| AQ | 1～5 | AQ07、AQ18 | **sum**（全量表） | 29 题；并计算子量表：`AQ_physical`、`AQ_verbal`、`AQ_anger`、`AQ_hostility`（各为对应题目子集之和，同用 sum） |
| ERQ_CR | 1～7 | 无 | **sum** | 6 题；分越高表示认知重评倾向越高 |
| ATTN | 1～5 | 无 | **mean** | 5 题；主要用于质控与注意题规则，一般不作为主维度的「特质分」与 PRDS 等并列解释 |

## 3. 反向计分公式示例

- **PRDS**（min=1, max=6）：反向后 = **7 − raw**  
- **AQ**（min=1, max=5）：反向后 = **6 − raw**

## 4. 理论分范围（全部填答、无缺失时）

| 量表 | 约略范围（sum 时） |
|------|-------------------|
| PRDS | 5～30 |
| PMD | 24～168 |
| AQ 总量表 | 29～145 |
| ERQ_CR | 6～42 |

子量表范围：身体 9～45，言语 5～25，愤怒 7～35，敌意 8～40（均为 5 点量尺下各子题之和）。

## 5. 与文献中「均分」的对应

若文献报告「各题均分」而非总分，可由 **总分 ÷ 题数** 得到，与总分单调对应；写论文时说明本研究采用系统所存总分/子分即可。
